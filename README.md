# CRYPTO RECORD BOT

## Description:
Basic bot written in golang to monitorize crypto currencies through telegram. 

---
## Basic commands

### /price

```
/price
```

```
/price ethereum
```

### /createalert

```
/createalert bitcoin > 40000.0
```

```
/createalert bitcoin < 28000.0
```

### /listalerts

```
/listalerts
```

### /deletealert

```
/deletealert bitcoin
```
